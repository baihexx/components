## 设计

非侵入式，尽量在不修改原有 DOM 结构的前提下实现功能，参见 *core/swiper*。

## 目标

### 近期目标

  编写尽量小的碎片化的组件，方便开发时灵活的组装，以应对多变的需求；

  什么叫小？参见 *core/form*；

  在这里，我们要先将之前使用 *nd-form* 的习惯从记忆中抹去，让一切简单、轻量。

### 中期目标

  根据实践反馈，持续不断地完善组件；

  参考借鉴外部开源项目，总结工作中的需求，在碎片化上稍稍封装一些偏重的组件。
